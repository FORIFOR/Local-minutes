# 手動チェックリスト（YouTube/Teams/Zoom 共通）

- OS 権限（マイク 3 トグル ON）
  - Windows 設定 > プライバシー > マイク > 「このデバイスのマイクへのアクセスを許可」「アプリによるアクセスを許可」「デスクトップアプリによるアクセスを許可」を全て ON
- ブラウザ権限（サイトごとのマイク許可）
  - Chrome/Edge で UI ドメインのマイク権限を「許可」に設定
- システム音共有
  - Chrome/Edge の「画面/タブ共有」で「音声を共有」を ON
- ヘッドセット推奨（ハウリング対策）
- UI テストページで WS URL を `ws://127.0.0.1:8765` に設定して「送信開始」
- YouTube で日本語音声を 1 分再生（字幕付き推奨）。Teams/Zoom でも同等の共有を実施
- 送信が継続し、`tests/asr_e2e/out/metrics.json` と `recv.wav` が生成されること
